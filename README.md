# Nombre del proyecto 

## 📋 Descripción

Es una aplicación multiplataforma enfocada en el área de logística. La idea es utilizar el sensor LiDAR de los celulares para escanear en tiempo real paquetes, productos y los espacios dentro de los camiones, con el objetivo de maximizar el uso del espacio disponible en el empaquetado de los productos.

## 📦 Estructura del Proyecto

Se utilizarán microservicios para mejorar el rendimiento de la aplicación, permitiéndonos tener pequeños servicios independientes que se comunicarán a través de APIs. Esto proporcionará una mayor flexibilidad, ya que cada servicio funcionará de manera autónoma. El principal motivo para utilizar microservicios es que React Native puede afectar el rendimiento si se desarrolla una aplicación híbrida y aumento en los errores de la aplicación afectando la experiencia del usuario.---
